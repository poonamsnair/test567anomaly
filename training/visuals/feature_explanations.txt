Feature Explanations for SHAP Analysis
==================================================

 0. num_steps: Number of steps in the trace
 1. duration_minutes: Total duration in minutes
 2. num_tools: Number of unique tools used
 3. num_agents: Number of unique agents involved
 4. avg_step_length: Average length of step content
 5. max_step_length: Maximum length of any step
 6. min_step_length: Minimum length of any step
 7. step_length_std: Standard deviation of step lengths
 8. unique_agents: Number of unique agents
 9. agent_entropy: Entropy of agent distribution
10. tool_entropy: Entropy of tool usage distribution
11. tool_usage_frequency: Frequency of tool usage per step
12. steps_per_minute: Steps executed per minute
13. question_length: Length of the original question
14. step_complexity: Average word count per step
15. repetitive_actions: Number of repetitive agent actions
16. tool_diversity: Diversity of tools used
17. step_consistency: Consistency of step lengths
18. agent_switching_frequency: How often agents switch
19. tool_usage_pattern: Hash of tool usage pattern
20. question_complexity: Word count of question
21. has_long_steps: Whether trace has very long steps
22. has_short_steps: Whether trace has very short steps
23. role_entropy: Entropy of role distribution
24. type_entropy: Entropy of step type distribution
25. consecutive_repeat_count: Number of consecutive repeated steps
26. tool_agent_overlap: Overlap between tools and agents
27. avg_step_duration: Average time per step
28. question_tool_match: Whether question mentions tools used
29. content_embedding_std: Standard deviation of content embeddings
